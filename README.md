# 🚀 Xeron Shop - O'yinlar uchun Raqamli Do'kon

**Xeron Shop** — foydalanuvchilarga turli xil o'yin ichidagi valyutalar (masalan, PUBG Mobile UC) va raqamli mahsulotlarni **X-coin** ichki valyutasi orqali sotib olish imkoniyatini beruvchi veb-platforma.

---

## ✨ Xususiyatlari
* 🎮 **Kategoriyalar:** O'yinlar bo'yicha mahsulotlarni filtrlash.
* 🪙 **X-coin Tizimi:** $1 \text{ UZS} = 1 \text{ X-coin}$ kursidagi ichki hamyon.
* 🛡️ **Xavfsiz Kirish:** Mahsulotlar va narxlar faqat ro'yxatdan o'tgan foydalanuvchilar uchun.
* 📱 **Responsive Dizayn:** Mobil va kompyuterlar uchun to'liq moslashuvchan interfeys.
* ⚡ **Tezkor Xarid:** SweetAlert2 integratsiyasi bilan tasdiqlash oynasi orqali xarid qilish.

---

## 🛠 Texnologiyalar
* **Backend:** [Django 5.0+](https://www.djangoproject.com/)
* **Frontend:** HTML5, CSS3, JavaScript
* **Ma'lumotlar bazasi:** SQLite (mahalliy), PostgreSQL (production)
* **Kutubxonalar:** SweetAlert2, FontAwesome, Google Fonts
